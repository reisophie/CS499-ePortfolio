Software Design and Engineering Enhancement

Artifact Description

The artifact I selected for this category is my CS 360 Daily Weight Tracker Android application. The app allows users to create a login, enter daily weight data, store progress, set a goal weight, and optionally receive SMS notifications when a goal is reached.

Enhancement Summary

  • Reduced repeated logic inside the activity classes
  • Created a reusable ValidationUtils class
  • Centralized validation for usernames, passwords, dates, weights, and phone numbers
  • Improved project organization and structure
  • Cleaned up parts of the app to make it easier to maintain

Why the Changes Matter

This enhancement helped improve separation of concerns and made the application more organized, readable, and maintainable. Instead of keeping validation and repeated logic scattered across different screens, I moved key responsibilities into reusable supporting code. These changes made the project stronger from a software engineering perspective and better reflect my current understanding of maintainability and code structure.
