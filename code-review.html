Code Review

Overview

This code review evaluates the original version of my CS 360 Daily Weight Tracker application. In the review, I identified opportunities for improvement in software design and engineering, algorithms and data structures, and databases. The code review helped guide the enhancements that I later completed throughout this capstone.

Code Review Video

Watch My Code Review Video

Main Findings

  • I identified software design issues related to too much logic being concentrated inside activity classes, especially ProgressActivity.
  • I found opportunities to improve validation, sorting, and progress calculations so the app would handle user data more carefully and more consistently.
  • I also identified ways to improve the database structure and organization, including reducing tight coupling between the activities and database helper and improving how user information was stored and handled.
