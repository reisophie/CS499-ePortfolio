# CS499-ePortfolio

This repository contains my CS 499 Computer Science Capstone ePortfolio.

## Overview

This ePortfolio highlights my growth throughout the Computer Science program and demonstrates my ability to improve an existing software artifact across three core areas of computer science:

- Software Design and Engineering
- Algorithms and Data Structures
- Databases

## Main Artifact

For this capstone, I selected my **CS 360 Daily Weight Tracker** application as the primary artifact for all three enhancement categories. I chose to use one artifact across the portfolio because it allowed me to show connected and meaningful improvement within a single project instead of presenting unrelated work from multiple courses.

## Portfolio Contents

- Code Review
- Software Design and Engineering Enhancement
- Algorithms and Data Structures Enhancement
- Databases Enhancement
- Professional Self-Assessment
